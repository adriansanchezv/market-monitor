How should the dashboards be navigated?

The existing Market Monitor is one large component. We need to decide how the new analytical dashboards plug in.

A

Top Tab Bar

Horizontal tabs across the top — Market Monitor | Monte Carlo | Regime | Sensitivity | Backtester | Risk | Sentiment | Correlation

Preview
Market Monitor Monte Carlo Regime Sensitivity Backtester Risk Sentiment Correlation
← Dashboard content fills below

Pros

  • Familiar pattern
  • One-click access

Cons

  • 8 tabs is crowded on small screens
B

Sidebar Navigation

Collapsible left sidebar with icons + labels. More room for tab names, works well as you add more tools.

Preview

🎲 Monte Carlo
🔍 Regime
🎚 Sensitivity
⏮ Backtester
⚠️ Risk
💬 Sentiment
🔗 Correlation

Pros

  • Scales to many tools
  • Easy to add more later

Cons

  • Takes horizontal space
C

Home Dashboard + Drill-in

A card grid home screen showing live summary tiles for each tool. Click a card to open the full dashboard.
